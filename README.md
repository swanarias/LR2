## Текст задания
Разработать объектно-ориентированную модель системы «Государственный аудит» на C# (xUnit, Moq). 
Обеспечить полное тестовое покрытие свойств, методов и состояний объектов. 
Настроить автоматизацию через GitHub Actions.

## Описание предметной области
Система государственного аудита предназначена для контроля целевого использования бюджетных средств. 
Процесс включает в себя планирование проверки (AuditPlan), сбор данных с серверов (AuditServer), 
анализ финансовых транзакций (FinancialTransaction) аудитором (Auditor) и формирование 
финального отчета (AuditReport), заверенного электронной подписью (SecurityToken).

## Список объектов системы
1. **AuditController** (Контроллер) — координация взаимодействия объектов.
2. **AuditServer** (Неодушевленный) — хранилище финансовых данных.
3. **SecurityToken** (Неодушевленный) — устройство для авторизации и подписи.
4. **Auditor** (Одушевленный) — должностное лицо, проводящее проверку.
5. **Accountant** (Одушевленный) — сотрудник организации, предоставляющий отчетность.
6. **FinancialTransaction** (Виртуальный) — запись о движении денежных средств.
7. **AuditPlan** (Виртуальный) — утвержденный график и объем работ.
8. **AuditReport** (Виртуальный) — итоговый документ с вердиктом.
